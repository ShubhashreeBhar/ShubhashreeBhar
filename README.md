## 🚀 Featured Projects:

**[feedtrust](https://github.com/ShubhashreeBhar/feedtrust)** &nbsp; `full-stack`
A crowdsourced review platform where people can post and trust community feedback. Built with React, Vite, Tailwind CSS, and PostgreSQL.

**[GrabPhisher-Model](https://github.com/ShubhashreeBhar/GrabPhisher-Model)** &nbsp; `graph deep learning`
Detects phishing transactions on a blockchain graph using Node2Vec embeddings and a Graph Convolutional Network, benchmarked against baseline models.

**[TGC-Model](https://github.com/ShubhashreeBhar/TGC-MODEL)** &nbsp; `research`
Transaction Graph Contrastive Learning for phishing node detection, learning node embeddings with DGL and PyTorch.

**[Trashium](https://www.trashium.com)** &nbsp; `full-stack + ML`
An incentivized recycling platform for households and collection crews, where pickup prices come from a machine learning model trained on regional scrap data. Built with Next.js, Supabase, and a scikit-learn pricing model.

**[Task Management (TaskTrack)](https://github.com/ShubhashreeBhar/Task-Management)** &nbsp; `MERN`
A full-stack task manager with JWT authentication, task CRUD, priority levels, due dates, and a statistics dashboard. Built with MongoDB, Express, React, and Node.
